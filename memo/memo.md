## 2022年12月9日

ステージのデータ構造どうする？
物理エンジン使うんなら一つ一つのオブジェクトをしっかり定義してあげた方がいいね
```lua
local data = {
    { -- level 1
        {
            type = "block",
            data = {
                {0, 0, 40, 720},
                {0, 680, 1280, 40},
                {1240, 0, 40, 720}
            }
        },
        {
            type = "goal",
            data = {
                {1000, 400, 40, 720}                
            }
        },
        ...
    },
    ...
}
```

コンポーネント指向に切り替えるので上はなし
```lua
return {
    -- object: player
    {
        player = {
            param = { 'need_physics_world' },
            100, 100
        },
    },
    ...
}
```
